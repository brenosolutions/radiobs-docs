# Ferramentas e Utilitários

Esta seção reúne scripts e utilitários essenciais para a administração do seu site.


## Redefinir Login do Administrador


> ⚠️ **Aviso de segurança:** Este procedimento deve ser usado **somente pelo proprietário do site**. Após o uso, remova o arquivo para evitar acesso indevido.

1. Baixe a ferramenta para redefinir o administrador: [Download](https://www.lugsoft.com.br/downloads/reset-admin.zip)
2. Descompacte o arquivo `reset-admin.zip` e envie `reset-admin.php` para o diretório raiz do servidor.
3. Para redefinir a login, acesse o seguinte URL no seu navegador: `https://seudominio.com.br/reset-admin.php`
4. Confirme na tela que o login foi redefinido para os dados de acesso padrão.
5. O script tenta se autodeletar. Se não sumir, apague manualmente via FTP ou Gerenciador de Arquivos.
6. Vá até a página de login e altere a senha para uma **senha forte** imediatamente.


## Verificar Compatibilidade do Servidor

1. Para verificar a compatibilidade do seu servidor, utilize nossa ferramenta de teste de requisitos necessários para instalação. [Download](https://www.lugsoft.com.br/downloads/reqs.zip)
2. Após o download, descompacte o arquivo reqs.zip e envie o arquivo resultante reqs.php para o diretório raiz do seu servidor.
3. Para verificar os resultados dos requisitos necessários, acesse o seguinte URL no seu navegador: `seudominio.com.br/reqs.php`
4. Analise os resultados exibidos pela página reqs.php e certifique-se de habilitar quaisquer módulos adicionais necessários conforme indicado no teste, para garantir o funcionamento adequado do sistema.
